一、运行环境
运行环境：Python 3.8 及以上版本
依赖库：仅使用 NumPy 基础数值库，未调用第三方有限元求解工具
依赖安装命令，在终端 / 命令提示符中执行：
pip install numpy
二、程序文件及功能说明
所有文件需放置在同一个文件夹内，具体文件作用如下：
main.py：主程序入口，串联所有功能模块，统一打印程序运行结果。
model.py：前处理模块，读取 JSON 格式模型文件，解析节点、单元、材料、边界条件与节点载荷，自动完成编号转换。
element.py：单元分析模块，计算单元几何参数、方向余弦、单元刚度矩阵、单元应力与轴力。
assembly.py：矩阵组装模块，生成 LM 对号矩阵，完成单元刚度矩阵到总体刚度矩阵的累加组装，同时判断矩阵对称性。
solver.py：求解模块，判断矩阵奇异性，采用缩减法处理位移边界条件，求解节点位移与约束反力。
postprocess.py：后处理模块，提取单元局部位移，计算并输出单元长度、方向余弦、应力、轴力。
case1_1d_bar.json：一维两单元杆结构模型输入文件（标准算例 1）。
case2_2d_truss.json：二维两杆桁架结构模型输入文件（标准算例 2）。
三、运行前置准备
将上述所有文件统一放在同一个文件夹中，不要拆分文件路径。
打开系统命令提示符（CMD）、PowerShell 或终端，使用 cd 命令切换到程序所在文件夹路径。
示例（Windows 系统）：cd D:\ 有限元作业 \2-3 桁架程序
确认 NumPy 库已成功安装，无环境报错。
四、运行命令
程序采用命令行传参形式运行，通用格式：python main.py 模型文件名.json
运行算例 1（一维两单元杆结构）：
python main.py case1_1d_bar.json
运行算例 2（二维两杆桁架结构）：
python main.py case2_2d_truss.json
五、程序输出内容说明
程序按照有限元计算流程依次输出内容，可用于结果核对：
前处理信息：打印模型名称、总节点数、总单元数、结构总自由度。
LM 对号矩阵：输出单元与全局自由度的映射矩阵，可手动校验自由度对应关系。
总体刚度矩阵：打印组装完成的总体刚度矩阵，同时输出矩阵是否对称、未施加边界条件时矩阵是否奇异。
求解结果：输出缩减后刚度矩阵的奇异性、全部全局节点位移、约束位置的约束反力。
后处理结果：逐单元输出单元长度、方向余弦、单元应力、单元轴力。
六、编号规则说明
程序采用两套编号体系，全程自动转换，无需手动修改：
外部输入文件（JSON 文件）：节点、单元、自由度均为从 1 开始编号（1-based），遵循作业规定格式。
程序内部计算：所有数组、矩阵、循环下标统一转换为从 0 开始索引（0-based），适配 Python 数组规则。
转换逻辑：程序读取 JSON 数据时，自动将外部编号减 1 完成转换。例如 JSON 中填写固定自由度为 1，程序内部识别为下标 0。
