一维对流扩散有限元求解程序运行说明
一、运行环境
Python 版本：3.7 及以上
支持系统：Windows、MacOS、Linux
绘图字体配置：代码内置微软雅黑、宋体，Windows 自带；Mac/Linux 缺失字体可自行修改代码字体参数
二、依赖安装
打开 CMD / 终端，执行命令安装依赖库：
pip install numpy matplotlib
三、运行操作步骤
解压全部文件至同一个文件夹，不可拆分文件
终端切换至文件所在目录
执行运行指令：
python advection_diffusion_fem.py
程序自动完成计算、打印数据、弹出绘图窗口并保存图片至本地文件夹
四、程序输出内容介绍
1. 控制台文字输出
Pe=0.1、Pe=3.0 两组工况的单元长度、扩散系数、SUPG 最优 α 参数
标准 Galerkin、迎风格式、SUPG 各自单独输出最大绝对误差
三种格式最大节点绝对误差汇总表，可直接复制粘贴进作业报告
Pe=3.0 标准 Galerkin 总矩阵对称性、正定性分析结果
附加题 nel=10/20/40/80 网格加密收敛误差数据
2. 自动生成图片文件
Pe_0.1_result.png：扩散占优工况，四条曲线对比精确解与三种数值格式
Pe_3.0_result.png：对流占优工况，直观展示 Galerkin 振荡、迎风数值扩散、SUPG 稳定高精度特性
grid_convergence.png：对数坐标误差收敛曲线，对比两种格式网格加密收敛性能
五、核心自定义函数（与作业要求接口完全一致）
alpha_supg (Pe)：求解 SUPG 最优稳定参数，处理 Pe 趋近于 0 除零问题
element_matrix (kappa, v, le, alpha)：生成两节点线性单元 2×2 刚度矩阵
solve_advection_diffusion (nel, L, v, kappa, alpha)：网格划分、总刚组装、边界条件施加、求解数值解与解析精确解
六、运行注意事项
核心离散、组装、求解逻辑手动编写，未调用专业有限元 / PDE 求解库，符合作业要求
精确解采用 np.expm1 计算，避免大 Pe 数下指数溢出数值不稳定问题
绘图中文乱码解决：系统安装微软雅黑 / 宋体，或修改代码 plt.rcParams 字体名称
重复运行会覆盖原有三张 png 图片，需要留存旧图请提前重命名
附加题网格加密计算自动执行，无需手动修改参数
