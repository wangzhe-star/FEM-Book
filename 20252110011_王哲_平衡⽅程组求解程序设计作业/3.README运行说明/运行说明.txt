# 有限元平衡方程组求解程序设计作业
## 1. 项目说明
本项目为《有限元平衡方程组求解》程序设计作业，基于Python实现：
1. 手写对称正定矩阵 LDLᵀ 稠密求解器
2. 病态矩阵残差、误差、条件数分析
3. 稀疏矩阵(COO/CSR/CSC) + Intel MKL PARDISO 大规模求解
4. 完全衔接2.3桁架有限元模块，完成整体计算流程
5. 二维Poisson方程有限元(T3三角形单元)求解与误差分析

## 2. 环境与依赖
Python >= 3.8
执行依赖安装：
pip install numpy scipy matplotlib pypardiso json5 psutil

## 3. 运行方式
1. 进入 `src` 目录
2. 执行主程序：
python main.py
3. 运行结果：
   - 终端输出所有算例日志、位移、应力、误差、耗时
   - 云图/误差图保存至 `results/figures`
   - 日志保存至 `results/logs`

## 4. 算例清单
1. 一维两单元杆桁架（2.3衔接算例）
2. 二维两杆桁架（2.3衔接算例）
3. 病态矩阵精度/残差分析
4. 多阶三对角对称正定矩阵测试
5. 非正定矩阵主元检测
6. Poisson方程有限元大规模稀疏求解(50×50/100×100/200×200网格)

## 5. 核心规则说明
- LDLᵀ分解、前代、回代**全部手动实现**，未调用线性求解库
- 支持非正主元检测与异常抛出
- 稀疏求解基于Intel MKL PARDISO工业级求解器
- 完全复用2.3作业：建模、单元刚度、总体组装、对号矩阵、后处理模块
